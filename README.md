# Plant Disease Detection System

AI-powered Plant Disease Detection using:

- React.js
- Django REST Framework
- TensorFlow/Keras
- SQLite

## Features

- Leaf image upload
- Disease prediction
- Confidence score
- Disease description
- Treatment recommendation
- Dashboard analytics
- Prediction history
- PDF report generation

## Technologies Used

### Frontend
- React.js
- Bootstrap
- Axios

### Backend
- Django
- Django REST Framework

### Machine Learning
- TensorFlow
- Keras
